Promise in JavaScript

January 24, 2020

In Technical terms, A Promise is a JavaScript object that links producing code and consuming code.

Now what is a Producing code and consuming code, Understand it with the help of this code snippet

When the code obtains result, it will get one of the two callbacks, namely:

  • If the result is success, callback will be myResolve( result value )

  • If the result is Error, callback will be myReject( error object )

Promise Object Properties

A JavaScript Promise object can be:

  1. Pending
  2. Fulfilled
  3. Rejected

While a Promise object is "pending" (working), the result is undefined.

When a Promise object is "fulfilled", the result is a value.

When a Promise object is "rejected", the result is an error object.
